# Описание
Репозиторий содержит численные данные и вспомогательные материалы к статье «Прогнозирование максимальных напряжений в системе вал–вкладыш с помощью нейронной сети», подготовленной для публикации в журнале «Научно-технический вестник информационных технологий, механики и оптики (ИТМО)».
Моделирование выполнено в Ansys Workbench 2022 R1
для трибофатической системы «вал–вкладыш», где:
- вал описан стандартным материалом Structural Steel из библиотеки Ansys;
- вкладыш выполнен из чугуна МОНИКА (Mo–Ni–Cu), разработанного на РУП «Гомсельмаш» совместно с ООО «НПО ТРИБОФАТИКА» (патент РБ № 15617).

Каждая строка выборки содержит результаты одного расчёта:

Поле | Описание |	Единицы
--------------------------------------
Fn | Контактная (нормальная)сила |	Н

Q | Изгибающая сила | Н

sigma_insert_max | Максимальное эквивалентное напряжение во вкладыше |	Па

sigma_shaft_max	| Максимальное эквивалентное напряжение в вале |	Па

Всего выполнено 146 численных экспериментов в диапазонах нагрузок: Fn = 0 ... 3000 Н, Q = -200 ... 200 Н. Набор данных использовался для обучения нейронной сети, прогнозирующей максимальные напряжения без повторного КЭ-моделирования.

Репозиторий предназначен для:
- обеспечения воспроизводимости численного эксперимента, описанного в статье;
- тестирования нейросетевых суррогатных моделей для задач трибофатики;
- сравнительного анализа распределений напряжений при различных нагрузочных сценариях.
